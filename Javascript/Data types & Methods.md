# Number  
All numbers are stored as `demimals`, but can be written with or without decimal places:
```js
typeof(45) // 'number'
typeof(32.21) // 'number'
```
Really **large** or **small** numbers can be written using `exponential notation`:
```js
let y = 123e5;    // 12300000  
let z = 123e-5;   // 0.00123
```
